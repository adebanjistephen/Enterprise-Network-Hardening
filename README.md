# Module 5 – Enterprise Network Security Hardening

This lab demonstrates the implementation of enterprise network security controls using Cisco Packet Tracer.

## Objectives

The lab focuses on securing enterprise network infrastructure through centralized authentication, access control, VLAN segmentation, Layer 2 security, and management plane protection.
